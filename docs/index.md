# Sumble Sales Brain

Eric's AE knowledge base. Built for fast reference on any device.

## Where to look

- **[Sales Brain](Sumble%20Sales%20Brain.md)** — main operational toolkit. ICP, plays, email rules, MCP tooling notes.
- **[Outbound Briefs](outbound-briefs/)** — strategic research briefs per target account.
- **[Battlecards](battlecards/)** — competitor objection handling. Clay, Common Room.
- **[Competitive Landscape](competitive-landscape.md)** — market positioning, customer power quotes, SWIM analysis.

## How to use this site

- **Search** — tap the magnifying glass top-right. Searches across every brief, battlecard, and the main brain.
- **Mobile** — fully responsive. Bookmark to your home screen.
- **Edit** — pencil icon top-right of any page jumps to the GitHub source.

## Adding a new brief

Drop a new `.md` file in `outbound-briefs/` following the [TEMPLATE](outbound-briefs/TEMPLATE.md) and naming convention `{target}-{anchor}-{YYYY-MM-DD}.md`. Push to `main` and it appears here within ~2 minutes.
